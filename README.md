# CPSC449 Group Project Final

## Final Project Team 14: Nicholas Caro, Justin Kim

### How to run 

```source bin/activate```

check

```pip3 list```

Install uvicorn

```pip3 install "uvicorn[standard]"```

Run uvicorn server

```uvicorn main:app --reload```

You can also run the server with

```uvicorn main:app --host 0.0.0.0 --port 80```
